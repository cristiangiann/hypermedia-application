\section{Variables}
	Some variables, considered important, have been chosen to be measured with the execution of the tasks and a subsequent survey for the users. The variables chosen to evaluate the website are the following:
	\begin{itemize}
		\item Effectiveness: tasks success rate;
		\item Efficiency: needed time to complete the tasks;
		\item Errors: wrong path or actions while completing the tasks;
		\item Satisfaction: users' feelings about the navigation experience;
		\item Disorientation: difficulty to realize what the user is seeing;
		\item Wandering periods: difficulty to realize the next steps;
		\item Design: structure of the pages.
	\end{itemize}
